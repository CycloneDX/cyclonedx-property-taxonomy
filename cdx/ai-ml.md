# `cdx:ai-ml` Namespace Taxonomy

This is the namespace for official CycloneDX properties specific to the Artificial Intelligence (AI)/machine Learning (ML) technology domain.

The official rules and processes apply - see [parent document](../cdx.md).

----

### `model` properties

The following table lists the current set of `model` properties.


| Property | Description |
|-----------|-------------|
| `cdx:ai-ml:model:parameter` | Used to describe a parameter used as input to a model for configuration or initialization purposes. |
| `cdx:ai-ml:model:hyperparameter` | Used to describe a hyperparameter provided as input to a model and used during generation. |
