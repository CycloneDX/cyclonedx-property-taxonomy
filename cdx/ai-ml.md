# `cdx:ai-ml` Namespace Taxonomy

This is the namespace for official CycloneDX properties specific to the Artificial Intelligence (AI)/machine Learning (ML) technology domain.

The official rules and processes apply - see [parent document](../cdx.md).

----

| Property | Description |
|-----------|-------------|
| `cdx:ai-ml:parameter` | Used to describe a parameter used as input to a model for configuration or initialization purposes. |
| `cdx:ai-ml:hyperparameter` | Used to describe a hyperparameter provided as input to a model and used during generation. |
